Fast, lightweight, extendable and easy to use library to handle 2d grid data.

## Installation

#### Using npm

```
npm i --save gridl
```

#### node.js

```javascript
var gridl = require('gridl');
```

#### ES6

```javascript
import gridl from 'gridl';
```

#### In a browser

```html
<script src="gridl.min.js"></script>
```
