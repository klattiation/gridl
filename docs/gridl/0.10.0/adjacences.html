Namespace: adjacences

adjacences

Predefined lists of adjacent positions relative to a certain position.
Properties:
Name Type Description
ALL Array.<Array.<number>> all direct adjacent positions (orthogonal + diagonal) in the order: left to right, top to bottom
ALL_CW Array.<Array.<number>> all direct adjacent positions (orthogonal + diagonal) in clockwise order
ALL_CCW Array.<Array.<number>> all direct adjacent positions (orthogonal + diagonal) in counterclockwise order
ORTHOGONAL Array.<Array.<number>> all orthogonal adjacent positions in the order: left to right, top to bottom
ORTHOGONAL_CW Array.<Array.<number>> all orthogonal adjacent positions in clockwise order
ORTHOGONAL_CCW Array.<Array.<number>> all orthogonal adjacent positions in counterclockwise order
DIAGONAL Array.<Array.<number>> all diagonal adjacent positions in the order: left to right, top to bottom
DIAGONAL_CW Array.<Array.<number>> all diagonal adjacent positions in clockwise order
DIAGONAL_CCW Array.<Array.<number>> all diagonal adjacent positions in counterclockwise order
