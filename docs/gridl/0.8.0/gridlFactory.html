Class: gridlFactory

gridlFactory(data) → {gridl}

new gridlFactory(data) → {gridl}

Creates a new gridl instance.

This is exported as the default function. It serves as a wrapper around gridl so that you don't have to use the `new` keyword each time. So instead of saying `new gridl(data)` you can just say `gridl(data)`. That's the only reason for gridlFactory.

Please don't care too much about the difference between gridl and gridlFactory. Just use it as `gridl(data)`.
Parameters:
Name Type Description
data Array A two dimensional grid array. Every row needs to have the same length.
